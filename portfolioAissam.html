Project 01:Optimizing Storage Space in a Seaport Using Genetic Algorithm

The aim of this project is to develop a genetic algorithm for optimizing container placement in a seaport. By taking into account key criteria such as container size and weight, as well as storage time, the algorithm improves storage space efficiency. Containers with shorter storage times are strategically placed closer to the exit, facilitating access and reducing processing times. This project demonstrates the application of advanced optimization techniques in a real logistics context, contributing to more efficient resource management in the maritime sector.
